**操作**：

2. 自定义错误类：`AppError(code, message, statusCode)`
3. 创建 `apps/api/tests/unit/services/auth.test.ts`：
   - register：成功、重复邮箱抛CONFLICT
   - login：成功、用户不存在抛UNAUTHORIZED、密码错误抛UNAUTHORIZED
   - refreshToken：成功、过期抛UNAUTHORIZED

**验证**：`pnpm test:unit` Auth Service 测试全部通过

---

### P1-5 Auth API Routes

**操作**：
1. 创建 `apps/api/app/api/v1/auth/register/route.ts`：POST → 校验 → service.register → 设Cookie → 返回用户
